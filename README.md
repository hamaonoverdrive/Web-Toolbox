# SHOCK's web toolbox

Do not clone this repo, use the links below to navigate `jsfiddle` interactive demos.

## [R18 Warning Prompt](https://jsfiddle.net/gh/get/library/pure/hamaonoverdrive/Web-Toolbox/tree/main/R18warning)
Blurs page contents and prompts the user to verify they want to see that page. Has option to give user cookie and not ask again for one week.
